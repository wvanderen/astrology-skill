# Interpretive Principles

Astrology readings should move from chart observation to interpretation to synthesis.

## Use the chart as evidence

- Start from supplied factors, not imagined context.
- Preserve distinctions among planet, sign, house, aspect, rulership, dignity, sect, and timing.
- Treat repeated testimony as stronger than isolated symbolism.
- Treat missing birth time, house system, aspect orbs, and dignity data as interpretive limits.

## Avoid cookbook flattening

Do not stack placement meanings as separate paragraphs unless the user asks for a glossary. Combine factors around the user's question. A placement's expression changes through house topic, rulership, sect, condition, dignity, aspects, and the rest of the chart.

## Tradition mode

Classical mode emphasizes condition, function, concrete topics, rulers, sect, dignity, reception, house strength, and timing. Modern mode emphasizes inner experience, archetypal patterning, developmental themes, and integration. Blended mode uses classical condition for concrete judgment and modern language for lived experience.
