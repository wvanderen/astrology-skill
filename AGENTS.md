## MANDATORY: Research Before Creating Skill Documents

Before starting any task that creates or materially updates reference documents for this skill, run a targeted research pass first.

- Search the existing repository for related modules, templates, roadmap notes, and naming conventions.
- When the document introduces or expands astrological doctrine, terminology, interpretive rules, or historical/traditional claims, perform an external source search when available and appropriate.
- Log the research pass with `td log`, including the task id, search focus, and sources or local files consulted.
- Do not draft new reference material from memory alone when a research search is possible.
