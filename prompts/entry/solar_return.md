# Solar return reading — entry fragment

Applies when `reading_type` is `solar_return`. Append this framing to
`prompts/entry/_reading.md`. This fragment adds **framing only**; it never
calculates the return chart or return moment.

## Chart-data emphasis

- The **return chart** for the year in question (Ascendant, chart ruler,
  angular planets, Sun placement, sect).
- The **natal chart** as the baseline the return modifies.
- The **return-year focus** carried in `user_question`.

## Framing for the reading

- Read the return chart in dialogue with the natal chart; do not treat the
  return as a standalone natal reading.
- Use only the factors supplied; do not derive the return moment or project
  future returns.
- Load `references/reading_types/solar_return.md`. Selection is owned by
  `SKILL.md`; this fragment only names it.
